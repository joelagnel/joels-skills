# joels-skills

Installable **agent skills** for [Claude Code](https://claude.com/claude-code),
[Codex](https://developers.openai.com/codex/), and [ChatGPT](https://chatgpt.com/skills),
following the [Agent Skills open standard](https://agentskills.io/): each skill is a plain
`SKILL.md` plus optional assets, so the same files work in every tool.

## Skills

| Kit | What it does |
|-----|--------------|
| **[HOW — hands-on workshop](handson-workshop/README.md)** | Turns any technical topic into a self-contained, auto-graded hands-on workshop grounded in real captured experiments: interview-calibrated depth, themed HTML (single page or multi-page wiki), diagrams, quizzes, and a final exam. Bundles the companion `handson-diagrams` skill. |

Each kit's README carries its full documentation: installation for all three tools,
quickstart, and how it works. ChatGPT-ready zips of every skill are published on the
[releases page](https://github.com/joelagnel/joels-skills/releases/latest).
